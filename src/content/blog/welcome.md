As a software architect, I wanted to create something that:

1. **Stands out** from typical portfolio websites
2. **Demonstrates technical abilities** through the implementation itself
3. **Provides an interactive experience** for visitors
4. **Appeals to fellow developers** who appreciate the command-line aesthetic

## Features

This terminal supports various commands to navigate and interact with the content:

- `ls` - List files and directories
- `cd` - Change directories
- `cat` - View file contents
- `open` - Open markdown files in a beautiful side panel
- `whoami` - Learn more about me
- And many more!

## Try It Out

Try exploring the filesystem:

```bash
ls
cd blog
ls
open welcome.md
```

All your changes are persisted to your browser's local storage, so you can create files, directories, and they'll still be there when you return!
